# BCIT - SSD
## Node Class
### Final Project  
<br />  

**- Instalation**  
On a terminal:
1. `git clone https://github.com/tonykielingbcit/node-project`
2. `cd node-project`
3. `npm i`  
<br />  

**- Run**
- set `.env` file accordingly
- `npm run dev` - dev mode, OR
- `npm start` - prod mode  
<br />  

**- Operation**  
Go to your brower and type `http://localhost:3003/` and navigate.
